# Section 5: Results analysis

Scripts and **notebooks** that produce the figures and tables used in the paper. This directory corresponds to **Section 5** (Results and Analysis) of the paper.

## Paper outputs (figures and tables included in the paper)

| Paper | File | Generated by |
|-------|------|---------------|
| **Figure 1** (Experiment Execution) | `figures/figure1_experiment_execution.pdf` | Diagram: export from `ExpExecution.drawio` (or use existing PDF); copy/rename to this name. |
| **Figure 2** (Metrics across topologies) | `figures/figure2_metrics_by_topology.pdf`, `.png` | `notebooks/Boxplots.ipynb` |
| **Table 1** (Topology summary) | — | Inline in paper (not generated here). |
| **Table 2** (Descriptive statistics) | `tables/table2_descriptive_statistics.tex`, `table2_descriptive_statistics.csv` | `notebooks/Descriptive_Statistics.ipynb` |
| **Tables 3, 4, 5** (KW + structural contrasts + Spearman) | `tables/table3_4_5_statistics.tex` | `notebooks/Structural_Groups_Analysis.ipynb` |

**Input:** `5_results_data/run_table.csv` (at repo root). **Outputs:** written to `5_results_analysis/figures/` and `5_results_analysis/tables/`.

## How to run

Use the project virtualenv so the notebooks have pandas, scipy, etc.

**From `5_results_analysis/`:**
```bash
source .venv/bin/activate
jupyter lab
```

**From repo root:**
```bash
source 5_results_analysis/.venv/bin/activate
jupyter lab
```

**Or** from `5_results_analysis/notebooks/` run `./START_JUPYTER_HERE.sh` (activates the venv and starts Jupyter with root `5_results_analysis`).

Then in Jupyter: open the **notebooks** folder and run (in any order):

1. **Boxplots.ipynb** → `figures/figure2_metrics_by_topology.pdf` and `.png`
2. **Descriptive_Statistics.ipynb** → `tables/table2_descriptive_statistics.tex` and `table2_descriptive_statistics.csv`
3. **Structural_Groups_Analysis.ipynb** → `tables/table3_4_5_statistics.tex`

If Jupyter was started from `5_results_analysis/`, the file tree shows `notebooks/`, `figures/`, `tables/` directly—open files from there (don’t use URLs with a leading `5_results_analysis/` in the path).
